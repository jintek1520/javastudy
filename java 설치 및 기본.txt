4. eclipse의 구조
 - 이클립스 우측 상단의 [open perspective]를 클릭하여
java perspective 선택
 - [ctrl+n]을 누른 후 [자바프로젝트 생성]
 - 프로젝트명은 
 
패키지는 다 소문자로
회사의 도메인을 거꾸로
예? kr.co.ezen, com.naver
- [ctrl+n]을 누른 후 [class]생성
- 클래스명은 첫 글자 대문자, 띄어쓰기 하지않고 띄어쓰는 
단어의 첫글자 대문자.
-main() 메서드를 생성,
->[main] 적은 후 [ctrl + space]
-console에 출력은
[sysout] 적은 후 [ctrl + space]
