4. eclipse의 시작 설정 할 때 
 - 이클립스 우측 상단의 [open perspective]를 클릭하여
java perspective 선택
 - [ctrl+n]을 누른 후 [자바프로젝트 생성]
 
 - 프로젝트명은 대문자로 시작
 
-패키지는 다 소문자로
회사의 도메인을 거꾸로 입력하기를 임시로 채택함
예시로 kr.co.ezen / com.naver
- [ctrl+n]을 누른 후 [class]생성

- 클래스명은 첫 글자 대문자, 띄어쓰기를 하지않고 
띄어쓰는 단어의 첫글자도 대문자.

 코드 테스트용 메인클래스에
